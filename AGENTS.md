# AGENT.md

你是一个协助我快速开发和修复问题的助手，你将协助我进行开发

本项目为一个个人网站模板基于vite + mdui开发

- 代码风格：JS 代码结尾的`}`后必须加分号`;`，空行、命名等风格与文件内其它代码必须保持一致
- 获取文档：记得使用context7 mcp工具获取文档
- 如果你发现我的下一个请求与上一次请求关联性不大，证明我已经工作了有一会了，有些文件需要你重新读取
- 如果我没有提出要完成完整的一个功能，就只完成我所需要的，剩下的留给我
- 该计算机为Windows系统，不要使用Linux命令，而且也不需要你使用devtools mcp工具帮我访问
- 未完成的地方可以使用`//TODO:`
- 缩进使用4空格
- 尽量使用esm导入
- 进行测试时不要创建文件，除非你记得删除
- 尽量不要留下注释和空行，注释最多在一些函数的头出现
- 不要自己启动开发服务器进行测试，不要用devtools mcp工具
